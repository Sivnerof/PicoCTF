In order to make sense of this, let's remove the useless functions that seemingly exist only to distract us and waste our time. The two functions, "esoteric1" and "esoteric2", only contain code that is commented out so we'll get rid of them. That leaves us with the following code that is much easier to read and understand:


The Python program never *directly* calls any of the defined functions, instead it prompts the user, suggesting that they type in "getRandomNumber" without quotations. Then the program uses the [(dangerous) eval function](https://nedbatchelder.com/blog/201206/eval_really_is_dangerous.html "Article on the dangers of Python's eval function") to interpret and execute the userâ€™s input as code.
